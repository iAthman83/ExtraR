The `extrar` package provides powerful tools for standardizing dataset structures, preparing "other" text responses for recoding, and performing complex survey data analysis.

## Installation

You can install the development version from GitHub with:

```r
install.packages("pak")
pak::pak("iAthman83/extraR")
```

## Getting Started

To learn how to use the package, please refer to the documentation pages:

- **[Data Preparation & Cleaning](https://iathman83.github.io/ExtraR/articles/data-cleaning.html)**: Guide on securely downloading data from KoboToolbox, standardizing raw data and loops, and handling "other" responses for recoding.
- **[Data Analysis](https://iathman83.github.io/ExtraR/articles/data-analysis.html)**: Guide on preparing the List of Analysis (LOA), running the grouping analysis pipeline, and exporting formatted Excel sheets.
